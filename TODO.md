# 学习Git中遇到的问题及思考解决
## 2018.04.22  

1. 【完成】怎么高效写-m "版本号信息"  
	项目名缩写+版本+简要信息，例如“职通车第二版新增TODO文档”可以写成“ZTC-0002 新增TODO文档”。

2. 【完成】如何简化pull/push命令，后面太长了  
	git pull/push origin master (首次推送时需要加-u进行关联)

3. 【完成】能否用其他文档代替word，github默认好像不支持word？  
	markdown可以直接在浏览器中显示内容
